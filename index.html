Zero A

Restituisci una lista contenente 1, 2, 3

Soluzione imperativa in Java


  public List kataZeroA(){
    List lista = new ArrayList<>();
    for(Integer i=1; i<=3; i++){
      lista.add(i);
    }
    return lista;
  }

Soluzione imperativa in PHP


    /**
     * @return array
     */
    public function kataZeroA(): array
    {
        $out = [];
        for ($i = 1; $i <= 3; $i++) {
            array_push($out, $i);
        }

        return $out;
    }

Zero E

Calcolare la media dei multipli di 8 compresi tra 1 e 100

Soluzione imperativa in Java


  public Double kataZeroE(){

    double sommaMultipli = 0;
    for(int i=8; i<=100; i+=8){

              $out += $i;

Zero H

Tra i numeri da 1 a 1000 trovarne uno qualunque divisibile per 41

Soluzione imperativa in Java


  public int kataZeroH(){
    List divisori = new ArrayList<>();
    for(int i=2; i<=100; i++){
      if(i % 41 == 0){
        divisori.add(41);
      }
    }
    return divisori.get(
      (new Random()).nextInt(divisori.size())
    );
  }

Soluzione imperativa in PHP


      /**
       * @return int
       */
      public function kataZeroH(): int
      {
          $out = [];
          for ($i = 41; $i <= 1000; $i += 41) {
              array_push($out, $i);
          }
          $randomicIndex = rand(0, sizeof($out) - 1);
          return $out[$randomicIndex];
      }

Soluzione funzionale in Java


    public int kataZeroH() {

    }

Zero K

Dall'elenco delle donne, restituisci una mappa in cui i nomi sono raggruppati per lunghezza

Soluzione imperativa in Java


  public Map<Integer,List<String>> kataZeroK(){
    List donne = ELENCO_DONNE;
    Map<Integer,List<String>> map = new HashMap<>();
    for(String donna : donne){
      int lunghezza = donna.length();
      if(map.containsKey(lunghezza)){
        List list = map.get(lunghezza);
        list.add(donna);
      }else{
        List list = new ArrayList<>();
        list.add(donna);
        map.put(lunghezza, list);
      }
    }
    return map;
  }

Soluzione imperativa in PHP


      /**
       * @param $womenNames
       * @return array
       */
      public function kataZeroK(array $womenNames): array
      {
          $out = [];
          foreach ($womenNames as $name) {
              if (isset($out[strlen($name)])) {
                  array_push($out[strlen($name)], $name);
                  continue;
              }
              $out[strlen($name)] = [$name];
          }
          return $out;
      }

Soluzione funzionale in Java


    public Map<Integer, List<String>> kataZeroK() {

    }
