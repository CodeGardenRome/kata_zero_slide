Concurrency

Side-effects e side-causes

In una funzione:

  • si intende come side effect un output non esplicito

  • si intende come side cause un input non esplicito

  • generalmente il termine side effects è utilizzato sia sia per input e output celati

Nel paradigma funzionale:

Gli input di una funzione devono essere argomenti

L'output deve essere ritornato dalla funzione

Un esempio con side effects


public class SideEffectsSum {
    int addendoA, addendoB, somma;

    public void somma(){
        somma = addendoA + addendoB;
    }
    public void setAddendoA(int addendoA) {
        this.addendoA = addendoA;
    }
    public void setAddendoB(int addendoB) {
        this.addendoB = addendoB;
    }
    public int getSomma() { return somma; }
}

High Order Function

Le high order function sono funzioni che accettano almeno due parametri: una funzione anonima (lambda) e una lista di elementi

Ogni elemento della lista viene passato in argomento alla lambda

La lambda controlla la trasformazione degli elementi in ingresso

Ogni HOF applica una trasfromazione differente

Map

Esempio di Map

Ho un vettore A che vale [1, 2, 3]

Ho una lambda che aggiunge 2 ad un numero

Se faccio
map( A, x -> x + 2 )
ottengo
[3, 4, 5]

Filter

Esempio di filter

Ho un vettore B che vale [1, 2, 3, 4]

Ho una lambda che restituisce true se il numero è pari, false altrimenti

Se faccio
filter( B, x -> x % 2 == 0 )
ottengo
[2, 4]

Reduce

Esempio di Redice

Ho un vettore A che vale [1, 2, 3]

Ho una lambda che, data la somma parziale e l'elemento corrente, produce una nuova somma

Se faccio
reduce( A, (s,x) -> s+x ) ottengo 6

Zero K

Dall'elenco delle donne, restituisci una mappa in cui i nomi sono raggruppati per lunghezza

Soluzione imperativa in Java


                          public Map<Integer,List<String>> kataZeroK(){
                            List donne = ELENCO_DONNE;
                            Map<Integer,List<String>> map = new HashMap<>();
                            for(String donna : donne){
                              int lunghezza = donna.length();
                              if(map.containsKey(lunghezza)){
                                List list = map.get(lunghezza);
                                list.add(donna);
                              }else{
                                List list = new ArrayList<>();
                                list.add(donna);
                                map.put(lunghezza, list);
                              }
                            }
                            return map;
                          }

Soluzione imperativa in PHP


      /**
       * @param $womenNames
       * @return array
       */
      public function kataZeroK(array $womenNames): array
      {
          $out = [];
          foreach ($womenNames as $name) {
              if (isset($out[strlen($name)])) {
                  array_push($out[strlen($name)], $name);
                  continue;
              }
              $out[strlen($name)] = [$name];
          }
          return $out;
      }

Soluzione imperativa in JavaScript


      export const ZeroK = donne => {
          let out = {};
          for(let i=0; i < donne.length; i++){
            let donna = donne[i];
            if( out.hasOwnProperty( donna.length ) ){
              out[donna.length].push(donna);
            }else{
              out[donna.length] = [donna];
            }
          }
          return out;
      }

Soluzione funzionale in Java

       */
      public function kataZeroL(array $womenNames): array
      {
          $out = [];
          foreach ($womenNames as $name) {
              array_push($out, strlen($name));
          }
          return $out;
      }

                          public List<String> kataZeroM(){

                            }
                            return list;
                          }
